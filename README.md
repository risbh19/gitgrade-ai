## Problem
Students often do not know how their GitHub projects appear to recruiters and mentors.

## Solution
GitGrade analyzes any public GitHub repository and generates:
- A quality score
- A professional summary
- A personalized improvement roadmap

## Architecture
- GitHub API for repository data
- Rule-based scoring for transparency
- AI-assisted feedback for mentorship
